# 音频自动降噪系统

## 项目简介
本系统为“信号与系统”课程期末项目，旨在实现一个音频自动降噪平台。用户可在前端上传音频文件（支持MP3、WAV、MP4等），点击降噪按钮后，系统自动处理音频并展示降噪前后对比及波形可视化，支持降噪后音频导出。

## 技术栈
- 前端：Vue3 + Vite
- 后端：Python（Flask，后续补充）

## 功能规划
1. 音频文件上传（多格式支持）
2. 降噪处理（后端API，Python实现）
3. 前后对比播放
4. 波形可视化
5. 降噪后音频导出

## 快速开始
### 前端启动
```bash
npm install
npm run dev
```
